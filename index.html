GLOSARIO PRIMER MODULO

HTML,CSS,GIT & GITHUB

  • FrontendParte de una aplicación web con la que el usuario interactúa directamente. Se construye usando tecnologías como HTML, CSS y JavaScript.
  • Fullstack Desarrollador que maneja tanto el frontend como el backend.

    • Programación Web Básica

    • API (Interfaz de Programación de Aplicaciones): Conjunto de funciones y procedimientos que permiten la interacción entre diferentes software y servicios.
    • HTTP (Hypertext Transfer Protocol): Protocolo de comunicación usado para transmitir datos a través de la web. Las solicitudes y respuestas entre el cliente y el servidor se basan en HTTP.

  • BranchVariante del código dentro de un repositorio. Las ramas permiten desarrollar características de forma independiente antes de integrarlas al código principal.
  • Commit Acción de guardar los cambios realizados en el código con un mensaje descriptivo.
  • Clone Copiar un repositorio remoto a tu máquina local.
  • Fork Copiar un repositorio remoto a tu cuenta de GitHub para poder trabajar en él sin afectar al repositorio original.
  • Git Sistema de control de versiones distribuido que permite llevar un seguimiento de los cambios en archivos y colaborar en proyectos de programación.

    • GIT & GITHUB

    • RepositorioAlmacenamiento centralizado de un proyecto de Git. Puede ser local o remoto (en plataformas como GitHub).
    • Pull RequestSolicitud para que los cambios realizados en una rama se integren a la rama principal de un proyecto, generalmente en GitHub.
    • PushAcción de subir los cambios locales de un repositorio a un repositorio remoto (por ejemplo, en GitHub).
    • GitHub Plataforma basada en la web para alojar repositorios Git, con características adicionales como seguimiento de problemas, gestión de proyectos, y colaboración en código.
    • Gitignore Archivo que especifica qué archivos o carpetas no deben ser seguidos por Git.
    • MergeAcción de combinar dos ramas diferentes en una sola.

  • <a>Enlace que permite navegar a otras páginas web.
  • <body> Contenedor del contenido visible de la página web.
  • <br> Salto de línea.
  • <div> Contenedor genérico usado para agrupar elementos.
  • EtiquetaElemento fundamental de HTML usado para definir partes de la página web, como párrafos, encabezados, imágenes, etc.
  • <form> Contenedor para formularios.
  • <h1><h1> a <h6>: Etiquetas para encabezados de distintos niveles, siendo <h1> el más importante y <h6> el menos importante.
  • <head>Contenedor de metadatos (información sobre la página), como el título o enlaces a hojas de estilo.
  • <html>Elemento raíz de un documento HTML.

    • HTML

    • HTML(HyperText Markup Language): Lenguaje estándar para estructurar contenido en la web.
    • <img> Inserta una imagen en la página.
    • <input> Campo de entrada para formularios.
    • <li>Elemento de lista.
    • <ol> Lista ordenada.
    • <p> Etiqueta para definir un párrafo de texto.
    • <span> Contenedor en línea usado para agrupar contenido pequeño.
    • <title> Define el título de la página, que aparece en la pestaña del navegador.
    • <ul> Lista no ordenada.

  • Box modelModelo de caja en CSS que describe cómo los elementos se estructuran en la página, incluyendo márgenes, bordes, rellenos y el contenido.
  • CSS Lenguaje usado para describir la presentación de un documento HTML, incluyendo diseño, colores, fuentes, y más.
  • ClaseUsada para seleccionar y aplicar estilos a un grupo de elementos. Se define con un punto antes del nombre (.mi-clase).
  • FlexboxModelo de diseño en CSS que facilita la creación de layouts responsivos y alineación de elementos.

    • CSS

    • GridGrid: Sistema de diseño en CSS para crear layouts complejos basados en filas y columnas.
    • IDSimilar a una clase, pero solo debe aplicarse a un elemento único. Se define con una almohadilla antes del nombre (#mi-id).
    • SelectorParte del código CSS que indica a qué elementos HTML se les aplicarán los estilos.
    • PropiedadCaracterística de un elemento que se va a modificar, como color, font-size, margin, etc.
    • ValorEl valor que se asigna a una propiedad, como red, 16px, 10px.
